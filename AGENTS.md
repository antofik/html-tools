### Description

This repository contains arbitrary tools that:
- represented as single self-contained HTML file
- do not require any external resources (e.g. JS/CSS scripts)

### UI Style
- White theme
- Buttons with svg icons with text as tooltips
- Maximum usability

### Tools
- **Diff View** allows to compare two text files
  Features:
    - allows pasting plain text
    - allows opening files from local disk
    - allows downloading compare result as self-contained HTML file
    - allows viewing diff as side-by-side and unified view
    - persistent settings
- **JSON Formatter** allows to prettify any JSON input
  Features:
    - allows to open json from file or paste it directly
    - persistent settings
    - view JSON as structured view, allowing to collapse/expand sections
    - array of simple objects are formatted in single-line 
    - allows to wrap text on/off
